# Vue School Project Repository
## Description

This repository is used for school assignments and contains a Vue.js-based project developed for educational purposes, practice, and academic tasks.

The project is based on the Sakai Vue Template developed by PrimeTek, and has been customized to meet learning and assignment requirements.

## Purpose

The purpose of this repository is to:

Serve as a workspace for school-related projects and assignments

Apply modern frontend development concepts

Practice using Vue.js and its ecosystem

Maintain a consistent and structured project setup for academic work

## Tech Stack

- Vue 3

- Vite

- PrimeVue

- Tailwind CSS (optional)

- SCSS

## Usage

This repository is intended for educational use only, primarily for completing and submitting school assignments.
The codebase may be modified, extended, or refactored as needed for learning purposes.
